# AI-Based-Face-Recognition-Attendance-System
AI-powered Face Recognition Attendance System built with Python, OpenCV, Streamlit, SQLite, and Machine Learning for automated real-time attendance tracking, analytics, and student management.
Engineered a real-time AI-powered computer vision system using OpenCV and a KNN classifier, achieving a 95%+ face
recognition accuracy for automated student identification.– Built an interactive Streamlit web application with role-based authentication, integrating live webcam streaming (We
bRTC) and a SQLite database for secure state management.– Designed a scalable model training pipeline and data augmentation strategy, generating automated CSV attendance
analytics and reducing logging errors by 30%
